# Especificaciones — Meta Quest 2, Quest 3 y Quest 3S

> Tema: fichas técnicas (specs) de los visores Meta/Oculus Quest 2, Quest 3 y Quest 3S.
> Fuente: imágenes de estudio `Oculus quest 2 Specs.webp` y `Oculus quest 3 Specs.webp` (transcripción).

## 1. Oculus Quest 2

| Característica | Especificación |
|---|---|
| **Procesador** | Qualcomm Snapdragon XR2 Octa-core con Adreno 650. |
| **Pantalla** | IPS LCD con resolución 1832×1920 píxeles por ojo y refresco de pantalla a 90 Hz. |
| **Memoria RAM** | 6 GB LPDDR5. |
| **Mandos** | 2 mandos 6DoF con pilas incluidas. |
| **Tracking** | Inside-Out. |
| **Almacenamiento** | 64 GB / 256 GB. |
| **Conectividad** | Bluetooth 5.0 LE, Wi-Fi b/g/n/ac/ax6/ay, conector jack de 3.5 mm y microUSB-C. |
| **Sistema operativo** | Oculus Mobile basado en Android 10. |
| **Dimensiones** | 190 × 105 × 115 mm y 503 gramos. |

## 2. Meta Quest 3 vs Meta Quest 3S (Key Specifications)

| Feature | Meta Quest 3 | Meta Quest 3S |
|---|---|---|
| **Body** | 184 × 160 × 98 mm | 191.5 × 102 × 142.5 mm |
| **Weight** | 515 gramos | 514 gramos |
| **Display** | Double LCD | Single LCD |
| **Lens** | Pancake | Fresnel |
| **Resolution** | 2064 × 2208 pixels (per eye) | 1832 × 1920 pixels (per eye) |
| **Field of view (FOV)** | 110° horizontal / 96° vertical | 96° horizontal / 90° vertical |
| **Battery capacity** | 4879 mAh | 4324 mAh |
| **Battery life** | 2.2 hours | 2.5 hours |
| **Storage** | 128 GB or 512 GB | 128 GB or 256 GB |
| **Launch price (base model)** | US$499.99 / £469.99 / €569.99 / ₹58,990 | US$299.99 / £289.99 / €329.99 / ₹24,500 |
